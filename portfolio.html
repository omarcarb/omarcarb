Where To Start

Inspiration

To start my portfolio website, I looked at a couple of inspirations from other designers and front end developers to get a sense of what I need for my website and have it laid out before my first line of code is even written. These two website from Matthew Williams and Naem Azam were my main inspiration since they both offer minimalist and responsive design

Design Direction

I really like William’s grid pattern for his projects but wanted to combine it with Azam’s neurmorphic design and color scheme. So I decided that my website will be a grid based layout with a neumorphism feel and look.

Low Fidelity Wireframes

Now that I had some inspiration, it was time for me to layout the foundation for the overall design and flow of the website.

High Fidelity UI Designs

For the final design, I decided to go with a neumorphism aesthetic to allow the website to stand out from others. As such, a lot of consideration was placed on shadows and soft colors.

Colors

Animations

To bring the website to life, I added a series of simple but lively animations to each element where appropriate.

Responsive Screen

After having the general layout and the designs laid out, I went forward with designing the website for several screen sizes. I primarily chose 3 screen sizes which were most common for website browsing.

Design to Code

At this point in the design process, I was satisfied with the end result. So it was time to make it all come to life with a little HTML, CSS and Javascript. But the process wasn’t as straight forward as it seems.

Final Thoughts and Conclusion

As my site begins to feature more projects, bug fixing will probably be the most important thing on my agenda. I will work continuously to improve my portfolio website by regularly identifying and fixing bugs to ensure a seamless user experience. In addition to maintaining its performance and reliability, I plan to enhance the website with new features and updates. These future enhancements will include adding interactive elements, optimizing for faster load times, and incorporating additional projects and case studies to showcase my evolving skills and expertise.
